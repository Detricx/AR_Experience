# AR_Experience

## Como funciona?
1. Terás de abrir o seguinte link por um dispositivo com uma qualidade razoável de câmara (Ex: Telemóvel)
   - https://detricx.github.io/AR_Experience
   - Para validares a qualidade, se desaparecer o loading, é sinal que irá funcionar.
2. Aponta a câmara para a imagem em questão e será suposto aparecer alguma ação para o efeito.

### 🎯 Funcionalidades

- Reconhecimento de imagem (logótipo da banda)
- Modelos 3D interativos:
  - Microfone
  - Guitarra elétrica
  - Teclado
  - Bateria
- Vídeo flutuante em AR
- Interface simples e visual

---

### 🚀 Usabilidade

1. Abrir o link do projeto no telemóvel  
2. Permitir acesso à câmara  
3. Apontar para o logótipo da banda  
4. Interagir com os elementos em realidade aumentada
