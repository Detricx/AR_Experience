# AR_Experience

## Como funciona?
1. Terás de abrir o seguinte link por um dispositivo com uma qualidade razoável de câmara (Ex: Telemóvel)
   - https://detricx.github.io/AR_Experience
   - Para validares a qualidade, se desaparecer o loading, é sinal que irá funcionar.
2. Aponta a câmara para a imagem em questão e será suposto ver um video que irá inicializar sozinho
   - Nota: Quando tiras a imagem do ecrã, ela irá colocar o video em pausa
