# 🎬 AR_Experience  
**Transforma posters de cinema em trailers com Realidade Aumentada**

---

## 📌 Sobre o projeto

O **AR_Experience** é uma aplicação WebAR que permite ao utilizador apontar a câmara do seu telemóvel para um poster de cinema e visualizar automaticamente o trailer associado.

O foco principal deste projeto é explorar a ligação entre o mundo físico e o digital, utilizando reconhecimento de imagem para transformar conteúdos estáticos (posters) em experiências dinâmicas e interativas.

A ideia central é simples: **um poster deixa de ser apenas visual e passa a ser uma porta de entrada para o conteúdo multimédia do filme**.

---

## 🎯 Objetivo do projeto

Desenvolver uma experiência WebAR acessível, intuitiva e interativa que:
- Demonstrar o potencial da Realidade Aumentada aplicada ao cinema  
- Permita visualizar trailers de forma imediata através de posters  
- Sirva como prova de conceito para aplicações futuras mais avançadas

- **Educação**  
  - Demonstração prática de tecnologias como AR e reconhecimento de imagem  

- **Desenvolvedores**  
  - Exemplo de integração de WebAR em projetos reais  

---

## ⚙️ Como funciona (fluxo)

1. Aceder ao link do projeto num dispositivo com câmara (telemóvel recomendado)  
   - https://detricx.github.io/AR_Experience  

2. Permitir acesso à câmara  

3. Apontar para o poster de cinema  

4. O sistema reconhece a imagem  

5. O trailer é apresentado em realidade aumentada (vídeo flutuante)

Uma solução como esta pode:
